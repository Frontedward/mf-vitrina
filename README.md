1. скопируй репозиторий к себе, открой его в своей IDE
2. открой терминал (ctrl + `) ⟶ ```npm install```
3. далее в консоле ```cd shared-components``` ⟶ ```npm run build```
4. после сборки в консоле ```cd ../``` ⟶ ```npm start```
5. после успешного старта открой в браузере ```http://localhost:3000/```
